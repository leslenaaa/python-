Sobre nosotros

En EducaWeys, nos dedicamos a ofrecer herramientas de educacion para ayudarte a alcanzar tus metas educativas y profesionales, a traves de las herramientas de nuestra página web, podras ir aprendiendo de manera continua y clara para que tengas un mejor desempeño.

Herramientas de Educación

Canva

Herramienta de diseño gráfico para crear presentaciones y materiales educativos.

⭐ 4.9 ℹ Canva es una empresa multinacional australiana de software patentado que proporciona una plataforma de diseño gráfico que proporciona herramientas para crear gráficos, presentaciones, postales, productos promocionales y sitios web para redes sociales.

Google Classroom

Herramienta muy buena para la organizacion de clases en linea.

⭐ 4.6 ℹ Google Classroom ayuda a los educadores a crear experiencias de aprendizaje interesantes que pueden personalizar, administrar y medir. Como parte de Google Workspace for Education, permite que los educadores mejoren su impacto y preparen a los estudiantes para el futuro.

Zoom

Plataforma de videoconferencias para clases y reuniones en línea.

⭐ 4.5 ℹ Zoom Video Communications es una empresa estadounidense fundada en 2011 por Eric Yuan, con sede en San José, California. Es conocida principalmente por su sistema de videollamadas y reuniones virtuales Zoom, disponible en computadoras de escritorio, computadoras portátiles, teléfonos inteligentes y tabletas.

Prezi

Herramienta muy practica para crear presentaciones bien elaboradas.

⭐ 4.7 ℹ Prezi es un programa de presentaciones para explorar y compartir ideas sobre un documento virtual basado en la informática en nube (software como servicio) ​ La aplicación se distingue por su interfaz gráfica con zoom, que permite a los usuarios disponer de una visión más acercada o alejada de la zona de presentación, en un espacio 2.5D.

Quizlet

Plataforma para crear tarjetas de estudio y practicar conceptos.

⭐ 4.8 ℹ Quizlet es una página web y aplicación de móvil educativa. Fue creado por Andrew Sutherland en octubre de 2005 y lanzado al público en enero de 2007. Quizlet entrena a los estudiantes a través de tarjetas de aprendizaje y varios juegos y pruebas.

Udemy

Ofrece cursos en línea sobre una amplia variedad de temas.

⭐ 4.6 ℹ Udemy es una plataforma de aprendizaje en línea. Está dirigido a adultos profesionales.​ A diferencia de los programas académicos MOOC conducidos por tradicionales cursos de trabajo creados por las universidades, Udemy utiliza contenido de creadores en línea para vender y así conseguir ganancias.​

Duolingo

Aprender idiomas nunca fue tan divertido e interactivo como en Duolingo.

⭐ 4.8 ℹ Duolingo ofrece cursos de idiomas gratuitos con ejercicios interactivos y seguimiento de progreso. ¡En nuestras lecciones cortas ganarás puntos y habilitarás nuevas unidades al mismo tiempo que desarrollas tus habilidades de comunicación en la vida real!

Python

Python es una herramienta de código, es facil de usar, muy eficiente, comoda y recomendad para la creación de proyectos pequeños y grandes.

⭐ 4.9 ℹ Python es un lenguaje de alto nivel de programación interpretado cuya filosofía hace hincapié en la legibilidad de su código. Se trata de un lenguaje de programación multiparadigma, ya que soporta parcialmente la orientación a objetos, programación imperativa y, en menor medida, programación funcional. Es un lenguaje interpretado, dinámico y multiplataforma.

Moodle

Esta es una plataforma para estudiantes de la universidad, cuenta con espacios de tareas, calendarios, etc.

⭐ 4.7 ℹ Moodle es un sistema de gestión de aprendizaje, gratuito y de código abierto escrito en PHP​ y distribuido bajo la Licencia Pública General GNU.​ Moodle se utiliza para el aprendizaje combinado, la educación a distancia, el aula invertida y otros esquemas de aprendizaje en línea en escuelas, universidades, lugares de trabajo y otros sectores.Se puede usar para crear sitios web personalizados con cursos en línea y permite complementos de origen comunitario.

GitHub

Ya tenemos herramienta de codigo y tambien necesitamos una para guardarlo, una muy buena opción para eso es GitHub.

⭐ 4.8 ℹ GitHub es una forja para alojar proyectos utilizando el sistema de control de versiones Git. Se utiliza principalmente para la creación de código fuente de programas de ordenador. El software que opera GitHub fue escrito en Ruby on Rails. Desde enero de 2010, GitHub opera bajo el nombre de GitHub, Inc.
