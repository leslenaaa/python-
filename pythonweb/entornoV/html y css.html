Resumen de HTML y CSS

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado que se utiliza para estructurar el contenido de las páginas web. Permite organizar textos, imágenes, enlaces, videos y otros elementos.

Está compuesto por etiquetas como <h1>, <p>, <img>, entre muchas otras.

¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje de estilos que se usa para definir la apariencia de un sitio web. Permite cambiar colores, fuentes, márgenes, tamaños, posiciones y más.

Con CSS podemos hacer que un sitio sea atractivo, moderno y fácil de usar visualmente.

¿Cómo se usan juntos?

HTML crea la estructura de una página, y CSS se encarga de su diseño visual. Se pueden combinar en un mismo archivo HTML usando la etiqueta <style> o enlazando un archivo externo de CSS.
